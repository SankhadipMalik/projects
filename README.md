# 📊 Exploratory Data Analysis (EDA) Projects

## 📌 Overview

This repository contains Exploratory Data Analysis (EDA) performed on real-world datasets such as **Bank Dataset** and **Diabetes Dataset**. The goal is to clean, analyze, and extract meaningful insights from raw data.

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Datasets Used

* `bank.csv` → Banking dataset for customer analysis
* `diabetes.csv` → Medical dataset for diabetes analysis

---

## 🔍 Key Features

* Data cleaning and preprocessing
* Handling missing values
* Outlier detection using statistical methods
* Data visualization (boxplots, distributions)
* Summary statistics and insights

---

## 📈 Analysis Performed

### 1. Data Cleaning

* Removed unnecessary spaces in column names
* Converted inconsistent values
* Replaced invalid values (e.g., 0 → NaN in medical data)

### 2. Statistical Analysis

* Mean, median, standard deviation
* Distribution understanding

### 3. Outlier Detection

* Used IQR method to detect and handle outliers

### 4. Visualization

* Box plots
* Distribution plots
* Feature analysis

---

pip install pandas numpy matplotlib seaborn
python bank_EDA.py
python Diabetes analyses.py
